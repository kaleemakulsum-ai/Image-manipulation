# Level 3 — Task 1: Image Manipulation

This is Task 1 of Level 3 for the Cognifyz Web Development Internship.

## What I Built

A CSS image gallery with a lightbox effect where thumbnails enlarge when clicked, and an automatic image slideshow that transitions between images every 3 seconds — both built using pure HTML and CSS only, no JavaScript.

## Features

- 6-image gallery in a responsive grid layout
- Lightbox effect on click using CSS :target selector
- Smooth zoom and fade hover effects
- Auto slideshow with CSS keyframe animations
- Fully responsive on mobile and desktop

## Files

- index.html
- style.css

## Tech Used

HTML5 and CSS3 only — no JavaScript, no frameworks
